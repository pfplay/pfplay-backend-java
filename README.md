## 1. Docker install

[Install link](https://docs.docker.com/engine/install/)

## 2. Clone source

```bash
% git clone 'https://github.com/pfplay/pfplay-backend.git'
```

## 3. Docker run
```bash
docker-compose up -d
docker-compose down
```
